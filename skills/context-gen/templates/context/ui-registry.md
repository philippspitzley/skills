# UI Registry

A living record of every UI component built in this project.
New components must match existing patterns unless the
developer explicitly approves a new pattern.

## How to Use

- Before building any UI component, read this file
- After building any UI component, append or update
  the relevant entry
- If a component type already exists, match its
  pattern exactly
- Run `/ui-snapshot` after building UI components to
  capture patterns automatically
